# Project Stay in the Loop
## CodeName: PACEKEEPER

目标：避免“压力→短视→掉队”循环，用可执行系统持续跟上时代。

## 目前确认的方向
1. ChatGPT Pro ($200/mo) 利用率最大化
2. 视频内容生产与增长（Remotion + ElevenLabs pipeline 迭代）
3. YouTube/X/TikTok 热门选题与分发体系
4. AI Workspace Kit（未来副业能力资产）

## 执行原则
- 每天固定进化时间（30/45/60 分钟择一）
- 输出优先于输入（每天至少一个可见产出）
- 项目绑定学习（优先与 GapBridge 主线联动）
