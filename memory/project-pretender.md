## 核心定位

中产润出信息服务商 — 卖信息差，不做移民中介，不碰法律，不做慈善。

## 底线

- 不骗人
- 筛选有认知提高空间的人
- 收多少钱做多少事
- 给建议让他们自己选择
- 信息差赚钱，明码标价

## 目标客群

- 中国中产（资产 50-500 万人民币）
- 从"岁月静好"中被打醒的人
- 有行动力但缺信息和方向
- 不知道从哪开始、不知道哪些信息是真的

## 商业模式

| 层级 | 服务 | 定价 |
|------|------|------|
| 免费 | 公开内容（Telegram 频道）— 唤醒认知、引流 | 免费 |
| 付费咨询 | 1对1 评估：资产/家庭/技能 → 推荐路径 | 按次/按小时 |
| 打包服务 | 具体国家落地指南 + 对接当地资源 | 按项目 |
| 长期关系 | 落地后生活咨询、资产配置 | 订阅/转介绍分成 |

## 渠道策略

- **主渠道**: Telegram 频道（匿名，润学圈主战场）
- **引流**: X 小号
- **人设**: 实操型、冷静、信息密度高

### 匿名措施
- 预付费物理 SIM 卡（现金购买，不关联身份）→ **待购买**
- ProtonMail 注册新邮箱
- Telegram 关闭"通过手机号找到我"
- X 小号用独立浏览器 profile
- 写作风格与主号（摸爬滚打的熊猫）隔离
- 收费走 Stripe 或加密货币
