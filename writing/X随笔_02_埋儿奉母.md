# X 随笔 #02：埋儿奉母

> 素材来源：2026-02-08 深夜思考 + 方法论序章
> 风格：如是我闻 + 调侃留白体

最近看到有人又把二十四孝翻出来了。

其中有个故事叫"埋儿奉母"：儿子为了省口粮给母亲，决定把自己孩子活埋。挖坑时挖到黄金，皆大欢喜。

小时候觉得这故事离谱。现在想想，更离谱的是——谁编的？

首先，肯定是个老人。年轻人编不出这种故事。

其次，肯定是个多子多孙的老人。只有一个孩子的人不会这么写——没有可损失的筹码。

多子多孙，损失一个孙子"不重要"。反正还有别的。

再看话术："孩子还可以再生，母亲只有一个。"

注意，是"母亲"，不是"父亲"。

在父系社会，父亲才是掌握资源的人。
但故事里，父亲消失了。

为什么？

因为用"母亲"更好操作。
反驳母亲 = 不孝 + 没良心。
父亲躲在后面，既是受益者，又没有责任。

妙不妙？

再看谁没有声音：儿媳妇。

她的孩子、她的子宫、她能不能再生——没人问。

最妙的是结尾：挖到黄金。

意思是：你孝顺，老天爷会奖励你。

翻译一下：你照做，不会有事的。

再翻译一下：别想太多，听话就行。

做不到埋儿？没关系。
那至少听话吧。天地君亲师，排好队。

这个要求就合理多了，对吧？

所以这故事的功能不是让你照做。
是让你觉得"至少我没那么过分"。
然后心安理得地服从。

妙啊。
